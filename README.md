# ai-recruiting

Автоматический скрининг резюме с hh.kz через Gemini AI. Расширение Chrome собирает резюме со страницы откликов, бэкенд оценивает каждое по вакансии (0–100 баллов), результаты видны в веб-дашборде.

---

## Быстрый старт

### 1. Получите Gemini API ключ

Перейдите на <https://aistudio.google.com/app/apikey> → создайте ключ → скопируйте.

### 2. Добавьте ключ в .env

Откройте файл `.env` в корне проекта и замените значение:

```bash
GEMINI_API_KEY=вставьте_ваш_ключ_сюда
```

### 3. Запустите сервер

Дважды кликните `start.bat` — он установит зависимости (если нужно) и запустит сервер.

Или вручную в терминале:

```
cd backend
python -m uvicorn main:app --host 127.0.0.1 --port 8000 --reload
```

Дашборд откроется по адресу: **http://localhost:8000**

### 4. Установите расширение Chrome

1. Откройте в браузере: `chrome://extensions/`
2. Включите **Режим разработчика** (переключатель справа вверху)
3. Нажмите **Загрузить распакованное**
4. Выберите папку `extension/` из этого проекта

---

## Как пользоваться

1. **Создайте вакансию** — в расширении нажмите «+», вставьте ссылку на вакансию hh.kz или введите текст вручную
2. **Откройте страницу откликов** на hh.kz (список резюме по вашей вакансии)
3. **Запустите скрининг** — нажмите кнопку «Старт» в расширении
4. **Смотрите результаты** — нажмите «Дашборд», откроется http://localhost:8000

В дашборде кандидаты разбиты по категориям:
- **Подходит** (80–100 баллов) — рекомендуется к собеседованию
- **Подумать** (66–79 баллов) — стоит рассмотреть
- **Отказ** (0–65 баллов) — не соответствует требованиям

Для каждого кандидата AI даёт комментарий и вопросы по пробелам в резюме.

### Массовые действия

- **Отказать** — расширение автоматически нажмёт кнопку «Отказать» на hh.kz для всех кандидатов из категории «Отказ»
- **В Huntflow** — откроет вкладки с резюме подходящих кандидатов
- **Экспорт** — скачает Excel-файл со всеми результатами

## Структура проекта

```
backend/        — FastAPI сервер + SQLite база данных
  main.py       — все API эндпоинты
  ai.py         — интеграция с Gemini
  database.py   — работа с БД
  static/       — веб-дашборд (HTML/JS/CSS)

extension/      — Chrome расширение
  popup.js      — интерфейс расширения
  content.js    — скрипт на странице hh.kz
  background.js — прокси для запросов к бэкенду
```
