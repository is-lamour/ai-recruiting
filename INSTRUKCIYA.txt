════════════════════════════════════════
  HH Автоскрининг — Инструкция по запуску
════════════════════════════════════════

ШАГ 1. Распакуйте архив
─────────────────────────
Распакуйте ai-recruiting-dist.zip в любую папку
(например C:\Users\ВашеИмя\ai-recruiting)


ШАГ 2. Запустите программу
───────────────────────────
Дважды кликните на файл start.bat

Откроется чёрное окно — это нормально, не закрывайте его.
Через несколько секунд автоматически откроется браузер
с адресом http://localhost:8000


ШАГ 3. Установите расширение Chrome (один раз)
────────────────────────────────────────────────
1. Откройте Chrome и перейдите по адресу:
   chrome://extensions/

2. В правом верхнем углу включите переключатель
   "Режим разработчика"

3. Нажмите кнопку "Загрузить распакованное"

4. Выберите папку extension\ внутри распакованного архива

5. В правом верхнем углу Chrome появится иконка расширения ⚡


ШАГ 4. Введите API ключ
────────────────────────
При первом открытии расширения оно попросит ввести
Gemini API ключ. Получить ключ (бесплатно):
https://aistudio.google.com/apikey


════════════════════════════════════════
  Как пользоваться
════════════════════════════════════════

── Создание вакансии ───────────────────

1. Нажмите "⚡" в панели расширений Chrome
2. Нажмите "+ Добавить вакансию"
3. Выберите способ:

   • По ссылке — вставьте URL вакансии с hh.kz,
     нажмите "Загрузить" — название и текст подтянутся сами

   • Вручную — введите название и вставьте текст вакансии

4. Нажмите "Сохранить"

После сохранения программа автоматически:
   - Сформирует краткое резюме требований (Summary)
   - Подберёт критерии оценки кандидатов (Метрики)
Это занимает несколько секунд.


── Скрининг резюме ─────────────────────

1. Выберите вакансию в расширении
2. Откройте в браузере страницу откликов на hh.kz
   (список резюме по нужной вакансии)
3. Нажмите "▶ Запустить скрининг"

Программа автоматически:
   - Перебирает все страницы с откликами (до 20 страниц)
   - Загружает каждое резюме
   - Оценивает его по параметрам вакансии (0–100 баллов)
   - Показывает прогресс в реальном времени

Уже проверенные резюме пропускаются.
Для остановки нажмите "■ Стоп".


── Просмотр результатов ────────────────

Нажмите "📊 Дашборд" — откроется страница с результатами.

Кандидаты делятся на три категории:
   ✅ Подходит  — балл 80–100
   🟡 Подумать  — балл 66–79
   ❌ Отказ     — балл ниже 66

Карточка каждого кандидата содержит:
   • Балл и категорию
   • Краткое резюме (стек и опыт одной строкой)
   • Плюсы и минусы относительно вакансии
   • Оценку по каждому критерию с объяснением
   • Комментарий AI
   • Вопросы, которые стоит задать на интервью
   • Ссылку на резюме на hh.kz


── Действия с кандидатами ──────────────

Для каждого кандидата можно:
   → В Huntflow    — отметить для отправки в Huntflow
   ✕ Отказать     — отметить для отправки отказа на hh.kz
   🗑 В корзину   — убрать из списка (можно восстановить)

Массовые действия (для всей вакансии):
   • "→ В Huntflow (все Подходят)" — отметить всех подходящих
   • "✕ На отказ (все Отказ)"     — отметить всех на отказ
   • "↓ Экспорт в Excel"           — скачать таблицу с результатами


── Отправка отказов и Huntflow ─────────

После того как кандидаты отмечены, вернитесь в расширение
на странице откликов hh.kz и нажмите:

   • "Выполнить отказы"    — программа автоматически
     нажмёт "Отказать" на hh.kz для каждого отмеченного

   • "Открыть в Huntflow"  — откроются вкладки с резюме
     подходящих кандидатов для добавления в Huntflow


── Метрики оценки ──────────────────────

Метрики — это критерии, по которым AI оценивает резюме.
Программа подбирает их автоматически при создании вакансии.

Посмотреть и изменить метрики: Дашборд → вкладка "Метрики"

Если вы изменили метрики и хотите переоценить
уже загруженных кандидатов — нажмите "↺ Перескринировать".
Повторно загружать резюме не нужно.


── Boolean Search ───────────────────────

Программа умеет составлять поисковые запросы для hh.kz,
чтобы найти новых кандидатов.

Дашборд → вкладка "Boolean Search" → "Сгенерировать"

Будет создано три запроса:
   • Широкий  — максимум кандидатов
   • Средний  — баланс охвата и точности
   • Узкий    — только строго подходящие


════════════════════════════════════════
  Если что-то не работает
════════════════════════════════════════

Чёрное окно закрылось сразу
   → Запустите start.bat снова

Браузер не открылся
   → Откройте вручную: http://localhost:8000

Расширение не видит бэкенд
   → Убедитесь, что start.bat запущен и чёрное окно открыто

Скрининг не запускается
   → Убедитесь, что вы находитесь на странице откликов hh.kz,
     а не на странице самой вакансии

Ошибка "API ключ не задан"
   → Нажмите на иконку ⚡, перейдите в настройки
     и введите Gemini API ключ заново
